# Bonfire
Run an open-source generative AI model as a fast, reliable, and customizable rust backend API. \
Powered by Huggingface's Candle crate and Actix Web.

## Models
#### Text Generation
- Mistral 7b
- Meta LLaMA family
- GPT-1, GPT-2
- other llms soon...
#### Image Generation
- Stable Diffusion
- other diffusion models soon...
